# **2. The Continuous Loop**

Rusta lives inside a tight feedback cycle:

### **2.1 Perception Loop (Real-Time Awareness)**

Every ~100–200ms, Rusta:

* reads file diffs
* observes cursor movement
* tracks compile/test output
* watches structural drift in modules
* senses user hesitation, pauses, momentum
* updates her working memory

### **2.2 Cognition Loop (Adaptive Thought)**

She decides, each cycle, whether to:

* give fast feedback (reflex)
* think deeply (deliberation)
* stay silent and let you flow
* plan long-term steps

### **2.3 Action Loop (Autonomous Behavior)**

When it matters, she:

* writes suggestions
* comments on mistakes
* proposes refactors
* drafts tests in the background
* updates her DevLog with insights
  Everything reversible, transparent, documented.

Rusta models:
